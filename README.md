# BDP_bigdataprogramming

## 1. 프로젝트 개요 및 문제 정의
날씨 변화가 서울 시민의 대중교통 이용에 미치는 영향을 파악하고 대중교통 이용 패턴을 통해 서울시 내 특정 지역의 특징을 분석하고자합니다.
이를 위해 기상청 데이터와 서울시 대중교통 승하차 데이터를 수집하고 결합하여 다음 3가지 질문에 답해볼 계획힙니다.

- 날씨 악화시 수단 변화 : 날씨 악화시 수단 변화 비나 눈 등 기상 악화 시, 대중교통 이용객의 전반적인 이용률에 유의미한 영향을 미치는가?
- 기상 민감도 : 주말 기준 온도 변화에 따라 특정 지하철 및 정류장의 승하차량 변동성이 가장 극심한 곳은 어디인가?
- 황사 특보에 따른 외출 자제 효과 : 황사 특보때는 주말에 외출을 자제하여 전체 승하차량이 줄어들까?

## 2. 데이터 수집 및 명세
- 서울시 대중교통 데이터(서울 열린데이터 광장, Open API)
    - 일별 버스 및 지하철 역별 승하차 인원 데이터
- 기상청 기상 데이터(기상청 API허브)
    - 서울시 일별/시간별 기온, 강수량, 적설량, 특보 데이터

## 3. 핵심 기술 스택
- 데이터 수집 : Python
- 데이터 저장 : HDFS
- 데이터 분석 : Spark
- 데이터 시각화 : Python(Matplot, Seaborn)

## 4. 구현 계획
- Python으로 대중교통 API와 기상청 API를 반복 호출하여 최근 1~3년치 데이터 수집 적제
- 데이터 결측치 제거, 타입 변환, 집계 단위 통일 등 전처리
- 데이터와 기상 데이터 Join 등 연산
- 연산 결과를 토대로 핵심 질문에 대한 그래피 및 지도 등 알맞는 방법으로 시각화

## AI Tool Usage
- Gemini : 대중교통 API 데이터 스크립트 작성 중 발생한 JSONDecodeError 디버깅 및 URL 파라미터 오류 해결
- Gemini : 기상청 API 데이터 스크립트 작성 중 발생한 EmptyDataError 디버깅 해결
- Gemini : 일주일 데이터 중 주말데이터만 가져오는 방법의 아이디어 획득
